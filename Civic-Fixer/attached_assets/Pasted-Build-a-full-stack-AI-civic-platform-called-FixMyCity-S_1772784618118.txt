Build a full-stack AI civic platform called **FixMyCity – Smart City OS**.

GOAL
Create a system where citizens can upload photos of city problems (garbage dumps, potholes, water leakage, broken streetlights, damaged infrastructure).
The platform uses **Google Gemini Vision AI** to analyze the image, identify the issue, determine severity, estimate impact, and guide citizens on how to report it.

The platform should look like a **modern civic-tech dashboard** and feel like a real startup product.

CORE FEATURES

1️⃣ AI PROBLEM DETECTION

User uploads an image.

Backend sends image to Gemini Vision API.

Prompt Gemini with:

"You are a city infrastructure inspector. Analyze this image and identify any civic problem. Respond only in JSON format."

Return:

{
problemType: string,
severity: High | Medium | Low,
description: string,
communityImpact: string,
authority: string,
estimatedFixTime: string,
actions: [4 steps citizens should take]
}

---

2️⃣ SMART AUTHORITY DETECTION

Automatically suggest which department handles the issue.

Examples:

Pothole → Road Works Department
Garbage → Municipal Sanitation
Water Leak → Water Board
Streetlight → Electricity Department

---

3️⃣ LOCATION DETECTION

Allow user to:

Use GPS location
OR drop pin on map

Show issue marker on map.

---

4️⃣ COMMUNITY ISSUE MAP

Display all reported problems on a city map.

Markers color coded by severity:

Red = High
Yellow = Medium
Green = Low

---

5️⃣ AI REPORT GENERATOR

Generate a formatted complaint.

Example output:

Subject:
Urgent Civic Issue – Road Damage

Body:
A pothole has been detected at [location].
Severity: High
Impact: Risk to vehicles and pedestrians.

Recommended action: Immediate road repair.

Allow:

Copy text
Download report
Share report

---

6️⃣ ISSUE UPVOTING

Allow citizens to upvote problems.

This highlights urgent problems.

---

7️⃣ HEATMAP ANALYTICS

Generate simulated heatmap of civic issues.

Shows areas with frequent problems.
